# Undisc0ver

Direct music storefront and download-gate platform for electronic artists.

Product name: `Undiscover`  
Domain target: `undisc0ver.com`.

## Local dev

```bash
npm install
npm run dev
```

Frontend: `http://localhost:5173`  
API: `http://localhost:3001`

## Production

Deployment files are included for Debian 12, Nginx, Docker Compose and MariaDB.

See [DEPLOY.md](DEPLOY.md).
